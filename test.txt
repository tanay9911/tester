git pull automatically performs git fetch and git merge into the local repository from the remote.
To create a new branch : git branch new-feature
To switch between branches : git checkout new-feature or git switch new-feature.
To merge branches :
git checkout main # switch to branch to which we want to merge
git merge new-feature.
To delete a branch :
git branch -d new-feature.
We use fork so as to keep a copy of og repo from git into our own account and we use this command:
 git remote add upstream <og_repo_url> # to keep the cloned repo upto date with the forked one.
Discard changes in your working directory :
git checkout -- <file(s)>
Unstaged changes from the staging area : git reset HEAD <file(s)>
git commit –amend : amend the last commit with new changes.
git revert <commit_hash> : revert a commit by creating a new commit.
git log
git show <commit_hash>
git diff <source_branch> <target_branch>
git remote add <remote_name> <repo_url>
git remote -v # lists remote repos
git remote remove <remote_name>
